# Explain - コード解説

## 目的
指定されたファイルの内容を読み、初心者にもわかるように解説する。

## 実行手順

1. **ファイル読み込み**: `$1` で指定されたファイルを Read で取得
2. **構造分析**:
   - ファイルの役割
   - 主要な関数/クラス
   - 依存関係
3. **解説作成**: 以下の観点で説明
   - このファイルは何をするものか
   - 主要な処理の流れ
   - 他ファイルとの関係

## 出力形式

```markdown
## [ファイル名] の解説

### 概要
このファイルは〇〇を担当しています。

### 主要な要素
- `functionA()`: 〇〇を行う
- `ClassB`: 〇〇を表現するクラス

### 処理の流れ
1. まず〇〇が呼ばれ
2. 次に〇〇が実行され
3. 結果が〇〇に返される

### 関連ファイル
- `xxx.ts`: 〇〇として使用
- `yyy.ts`: 〇〇から呼び出される
```
