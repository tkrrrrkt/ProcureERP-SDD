# Cleanup - ファイル整理

## 目的
不要なファイル（ログ、一時ファイル、ビルド成果物など）を検出し、整理を提案する。

## 実行手順

1. **対象ディレクトリ確認**: `$1` または現在のディレクトリ
2. **不要ファイル検出**:
   - `*.log` ログファイル
   - `node_modules/` 依存関係（再インストール可能）
   - `dist/`, `.next/` ビルド成果物
   - `.tmp`, `.cache` 一時ファイル
3. **サイズ計算**: 各カテゴリの合計サイズを算出
4. **提案**: 削除候補をリスト化（実際の削除はユーザー確認後）

## 出力形式

```markdown
## 整理提案

| カテゴリ | ファイル数 | サイズ | 削除コマンド |
|---------|-----------|--------|-------------|
| ログ    | 5         | 2.3MB  | `rm *.log`  |
| キャッシュ | 12      | 150MB  | `rm -rf .cache` |

### 推奨アクション
上記を確認し、削除する場合はコマンドを実行してください。
```

## 注意
- このスキルは**提案のみ**行い、自動削除はしない
- `.git/` や `src/` など重要なディレクトリは対象外
